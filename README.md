🔐 Password Strength Checker

A simple and user-friendly Password Strength Checker built using Python. This application evaluates the strength of a password based on common security standards and provides feedback to help users create stronger passwords.

🚀 Features
🔍 Password Analysis
Checks password strength in real-time
📊 Strength Classification
Weak ❗
Medium ⚠️
Strong 💪
✅ Validation Rules
Minimum 8 characters
At least one uppercase letter
At least one number
At least one special character
💡 Smart Feedback
Suggests improvements for weak passwords
Displays missing criteria
🎨 Simple GUI
Built with Tkinter
Clean and minimal interface
Hidden password input for security
🛠 Tech Stack
Python
Tkinter – GUI development
Regex (re module) – pattern validation
